# Night Blossom visual and motion specification

## Geometry

- Launch: 1548×970
- Minimum: 1140×720
- Custom title bar: 30 px
- Sidebar: 204 px normally; 104 px compact
- Header zone: 106 px
- Bottom changelog/log: 176 px expanded; 42 px collapsed
- Primary corner radii: 8, 14, and 20 px

## Motion

- Button hover lift: 1 px / 140 ms
- Card hover lift: 2 px / 145 ms
- Press scale: 0.985 / 70–75 ms
- Active route indicator: 150–160 ms
- Primary sheen: one 420 ms pass per hover entry
- Page entry: 190 ms fade
- Branch sway: 8–9.6 seconds each direction
- Petals: eight maximum, staggered 18–27 second paths

Reduced Motion disables ambient and nonessential transitions. Glass Effects disables expensive layer effects while retaining opaque/translucent panel structure and borders.
