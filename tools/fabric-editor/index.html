    What This Does

    This prototype is for manual cleanup of generated or handmade FH6 JSONs. It loads the JSON as editable FH6 vinyl layers, lets you move/delete/duplicate/reorder shapes, then exports an FH6 type-code JSON for the handmade importer.

    Basic Workflow

    1. Click Import JSON and choose a generated final JSON or handmade/exported FH6 JSON.
    2. Click visible parts of the design to select the top visible layer.
    3. Drag selected layers to move them, or drag a box to select multiple layers.
    4. Use Delete, Duplicate, Forward, Back, or arrow nudges to clean up visible mistakes.
    5. Click Export FH6 JSON for the handmade importer, or Export Normal Import JSON when the design still only uses generated rectangles/ellipses.

    Top Buttons

    • Import JSON: Loads generated legacy JSON or handmade/exported FH6 JSON.
    • Export FH6 JSON: Saves the current editable layer stack as FH6 type-code JSON. Use the handmade importer. Supports the full shape library.
    • Export Normal Import JSON: Saves a normal generated-style JSON for the fast normal importer. This only works when every layer is compatible with generated rectangles/ellipses.
    • Save Project: Saves the current editor state for later editing.
    • Load Project: Reopens a saved editor project.
    • Visible-only select: Makes clicks prefer actual visible pixels instead of a shape's full bounding box.
    • Fit Design: Zooms to the loaded layer stack.
    • Reset View: Recenters and zooms the canvas back to the FH6 bounds.
    • Undo / Redo: Steps backward or forward through editor changes.

    Layer Editing

    • Select only visible pixels: Clicks the actual visible shape area instead of the whole bounding box.
    • Nudge: Sets arrow-key movement amount. Hold Shift for a bigger nudge.
    • Delete: Removes selected layer(s).
    • Duplicate: Copies selected layer(s).
    • Forward / Back: Changes layer order.

    Shape Library

    • Pick a category, then click a shape tile to add it to the canvas.
    • Click the small corner star to add a favorite. In Favorites view, the same corner button removes it.
    • Search accepts shape names, category names, numbers, and type codes.
    • New shapes use the last manually selected color. Enable overlay color sampling to color new or edited shapes from the reference image.

    Reference Overlay

    Add a source image as a non-exported overlay. Adjust opacity and size to compare your edited shapes against the intended artwork. The overlay is never written into the FH6 JSON.

    Shortcuts

    • Mouse wheel: Zoom.
    • Middle/right drag: Pan.
    • Delete / Backspace: Delete selected layers.
    • Ctrl+D: Duplicate selected layers.
    • Ctrl+Z / Ctrl+Y: Undo / redo.
    • Arrow keys: Nudge selection.
    • Shift + Arrow keys: Larger nudge.
    • [ / ]: Move layer order backward / forward.

    Important Limits

    This is not an arbitrary SVG/path converter. It edits FH6 layers and exports FH6 JSON. If a shape type exists in FH6, it is preserved by type code. Arbitrary freehand SVG paths are not supported.
