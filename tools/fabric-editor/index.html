    FH6 Vinyl Editor Help

    What This Editor Does

    This editor works on FH6 vinyl JSON, not pixels. It lets you import generated or handmade JSON, edit the vinyl layers visually, trace over a source image, use the FH6 shape library, then export JSON back into the KFPS import workflow.

    Think of it as a safer desktop workspace for vinyl layer cleanup before you import into FH6.

    Basic Safe Workflow

    1. Import JSON: load a generated final JSON or a compatible handmade/exported FH6 JSON.
    2. Add Overlay if you want to trace over the original source image.
    3. Select / Move: click visible vinyl shapes, box-select groups, or select rows in Layers.
    4. Clean up: move, stretch, skew, rotate, recolor, delete, duplicate, group, hide, lock, or reorder layers.
    5. Use Export Check: confirm whether the design can use Generated export or needs Handmade export.
    6. Export: use the export button that matches the JSON type you want to import.

    Tools On The Left

    • Select / Move: normal editing mode. Click a shape, drag it, box-select, resize, rotate, or edit layer values.
    • Shapes: choose shapes from the library and place them into the current viewport.
    • Dropper: pick a color from a vinyl shape, or from the source overlay when no shape is under the click.
    • Guides: draw editor-only guide lines and helper grids for alignment. Guides do not export as vinyl layers.
    • Overlay: add, hide, scale, or remove a source image used for tracing and color sampling.

    Shape Library

    • Use the category dropdown to browse shapes in an FH6-like order.
    • Use search for shape names, numbers, families, or type codes.
    • Click a tile to place that shape in the current viewport.
    • Use the favorite button on a tile to keep common shapes near the top.
    • New shapes use the active color unless live overlay sampling is enabled.

    Layers, Groups, Locking, And Visibility

    • Top layer rows draw over lower layer rows.
    • Use Forward and Back to change draw order.
    • Use Group for editor-only organization. Groups help select, hide, or lock related layers without changing export structure.
    • Hide Group hides grouped layers while editing. Hidden vinyl layers are still part of the project unless deleted.
    • Lock Group prevents accidental edits to grouped layers.
    • Visible-only select makes clicks prefer actual visible pixels instead of huge bounding boxes.

    Color And Overlay Sampling

    • The color panel edits the active layer color and alpha.
    • Saved colors work like a small paint palette for colors you reuse often.
    • The overlay image is only a reference. It is never exported into JSON.
    • Sample overlay under selection recolors selected layers from the source overlay beneath their current shape area.
    • Live auto color from overlay updates color while you move or resize a shape over the overlay.
    • Dominant color is usually better for flat decals. Average color is useful for softer shaded regions.

    Guides, Grid, And Snapping

    • Guides are editor-only alignment helpers. They do not export.
    • Select Guides, then left-drag on the canvas to draw a guide line.
    • Right-drag or middle-drag pans the canvas without leaving guide mode.
    • Choose Free line, X axis / horizontal, or Y axis / vertical for guide drawing.
    • Enable the grid when you need repeated spacing or coarse layout checks.
    • Hold Control while moving, scaling, skewing, or reshaping a vinyl layer to snap to guides or grid lines.
    • For resizing/stretching, the side you drag is the side that should snap. The opposite side stays anchored as much as possible.
    • Angled guide snapping can rotate the shape to match the guide. Horizontal/vertical guides can bring it back to straight alignment.
    • Use Clear Guides when helpers get in the way. This removes guides only, not vinyl layers.

    Exports

    • Export for Handmade: full FH6 shape-library export. Use this for hand-edited designs or anything using non-generated shapes.
    • Export for Generated: normal generated-style export. Use this only when the design still contains generated-compatible rectangle/ellipse layers.
    • Save Project: saves editor state for later editing. This is not the same as an FH6 import JSON.
    • Load Project: reloads a saved editor project.

    Shortcuts And Mouse Controls

    • Mouse wheel: Zoom.
    • Middle/right drag: Pan.
    • Drag empty canvas in Select mode: Box-select layers.
    • Control while dragging/editing: Snap to active guides or grid helpers.
    • Delete / Backspace: Delete selected layers.
    • Ctrl+D: Duplicate selected layers.
    • Ctrl+Z / Ctrl+Y: Undo / redo.
    • Arrow keys: Nudge selection. Hold Shift for larger nudges.
    • [ / ]: Move layer order backward / forward.

    Important Limits

    This is not a raster image editor. It edits FH6 vinyl layers and exports FH6 JSON. Arbitrary freehand SVG paths are not supported.

    If an exported design imports incorrectly, first check whether you used the matching export/import path, then save and reload the group in FH6 before judging shape-code imports.
